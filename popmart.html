Building a Predictor Tool For POPMART's POP NOW Function

Duration: 2 months
View Product Page

PopMart POP NOW
Predictor Tool

Problem?

POP NOW users want to be confident in which blind box to pick.

POP NOW is a function from POP MART that allows users to open blind boxes virtually. While POP NOW allows users to know a set number of which specific items are not found in a box, users don't have the time to calculate the likelihood of their choice item appearing in a blind box. Users only find out which item they got after purchase.

Existing page

Hover over the points below to see highlights
1

Users are only given 5 minutes to reserve a blind box.

2

Each blind box will give hints to which items are not in a specific box.

With the limited time, users often find themselves spending more to get their choice item.

Goals

To create a quick and easy tool for users to pick blind boxes

Keep track of the user inputs for each blind boxes in an organized manner.

Create visuals that dynamically updates based on user input.

Let users easily customize the blind boxes by removing specified items.

Rethinking the flow

This new tool should not stray far from POP NOW's flow. As mentioned on their official website, their flow is three simple steps.

POP NOW's flow

The predictor tool will also be a three step flow that is similar to the official.

Predictor Tool Flow

Solution

A platform that generates predictions based on user input, guiding them on which blind box to pick.
